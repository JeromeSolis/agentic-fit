What the benchmark found

      
    
        
  1. The right library depends on the model. In six of seven head-to-head categories the cheapest reliable library changes from one model to the next. Only command-line parsing is unanimous.
    2. 
        
  2. The choice changes the bill. Among libraries that all work, cost varies a median of 1.7 times, and up to 24 times at the extreme.
    3. 
        
  3. Popularity does not tell you the answer. Agreement between a library's PyPI download rank and its cost rank for the agent is 54 percent, barely above chance.
    4. 
        
  4. The per-model signal is worth considering. A single default for every model costs about 1.25 times more, and sometimes fails outright where a model-specific choice succeeds.

Why this exists

      
Agents lean on the libraries and documentation that people build and maintain, yet the signals that once rewarded that work do not capture the value agents now pull from it. agentic-fit is a small, repeatable reading of which libraries agents actually use well, so a conversation about a fairer incentive loop between agents and the people building what they depend on can rest on measurement rather than intuition.
